Command-line Interface#

cookie-composer#

Render templates using composition.

cookie-composer [OPTIONS] COMMAND [ARGS]...

Options

--version#

Show the version and exit.

add#

Add the template or configuration PATH_OR_URL to an existing project.

cookie-composer add [OPTIONS] PATH_OR_URL [CONTEXT_PARAMS]...

Options

-v, --verbosity <LVL>#

Either CRITICAL, ERROR, WARNING, INFO or DEBUG

--no-input#

Do not prompt for parameters and only use cookiecutter.json file content. Defaults to deleting any cached resources and redownloading them.

-c, --checkout <checkout>#

branch, tag or commit to checkout after git clone

--directory <directory>#

Directory within repo that holds cookiecutter.json file for advanced repositories with multi templates in it

-f, --overwrite-if-exists#

Overwrite the contents of the output directory if it already exists

-s, --skip-if-file-exists#

Skip the files in the corresponding directories if they already exist

--default-config#

Do not load a config file. Use the defaults instead

-d, --destination <destination>#

The directory to add the templates to. Defaults to the current working directory.

--accept-hooks <accept_hooks>#

Accept pre/host hooks

Options:

yes | ask | no | first | last | all | none

Arguments

PATH_OR_URL#

Required argument

CONTEXT_PARAMS#

Optional argument(s)

auth#

Authenticate cookie-composer to a service.

cookie-composer auth [OPTIONS] COMMAND [ARGS]...

login#

Authenticate to a service.

cookie-composer auth login [OPTIONS]

Options

-p, --git-protocol <git_protocol>#

The protocol to use for git operations

Options:

https | ssh

-h, --service <service>#

The host name of the service to authenticate with

-s, --scopes <scopes>#

Additional authentication scopes to request

--with-token <with_token>#

Read token from standard input

token#

Print the auth token cookie-composer is configured to use.

cookie-composer auth token [OPTIONS]

Options

-h, --service <service>#

The host name of the service to authenticate with

cookie-composer create [OPTIONS] PATH_OR_URL [CONTEXT_PARAMS]...
